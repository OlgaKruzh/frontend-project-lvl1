## Установка

```sh
$ make install
$ make link 
$ make brain-games # run brain-games
$ make brain-even # run brain-even
$ make brain-calc # run brain-calc
$ make brain-gcd # run brain-gcd
$ make brain-progression # run brain-progression
$ make brain-prime # run brain-prime

```
